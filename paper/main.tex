\title{Mem0-Cognitive: Emotion-Weighted Forgetting and Sleep Consolidation for Adaptive LLM Memory}

\begin{document}

\maketitle

\begin{abstract}
Current memory-augmented Large Language Models (LLMs) typically accumulate
context passively, which causes retrieval precision to degrade as the memory
store grows. We introduce \textbf{Mem0-Cognitive}, a research prototype that
couples three cognitively-motivated mechanisms on top of a standard vector
memory layer: (1) \textbf{Emotion-Weighted Forgetting} \textemdash\ a modulation
of the Ebbinghaus decay curve by a per-utterance emotional intensity term
extracted via zero-shot LLM prompting, with a salience gate that protects
high-intensity memories from automatic deletion; (2) \textbf{Offline Sleep
Consolidation} \textemdash\ a clustering plus LLM-based abstraction step that
compresses redundant episodic traces into consolidated entries; and (3)
\textbf{Adaptive Parameter Tuning} \textemdash\ a lightweight
top-$k$-weighted-averaging heuristic over observed retention performance that
personalizes per-user retention parameters (an earlier framing as
Gaussian-Process Bayesian Optimization is retained as future work but not
claimed as an implemented result in this manuscript). On the proposed
CognitiveBench long-dialogue simulator and on LoCoMo, the combined system
reduces noise ratio and improves Retention@10 relative to vanilla RAG and Mem0
baselines, with the sleep-consolidation component responsible for the largest
single gain. Code supporting the results in this paper is available at the
anonymized companion repository linked in the submission materials.
\end{abstract}
